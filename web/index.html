Module

Jedes Modul deckt einen spezifischen Planungsbereich der Gebäudeautomation ab.

01
Verfügbar

Schaltschrank-Dimensionierung

Abschätzung der erforderlichen Schaltschrankgröße auf Basis von Anlagengruppen, Datenpunkten und Kundenvorgaben. Ausgabe: Abmessungen, Anzahl, Kostenschätzung und schematische Aufbauzeichnung.

  • Wand- und Standschrankvarianten
  • Kabeleinführungszone h_ke nach VDE 0298-4
  • Montageplatte: freie Wahl oder getrennte Zonen
  • Kostenschätzung inkl. DL-Felder
Modul öffnen → Zeichnung ansehen
02
Geplant

Einspeisezone & Klemmenzone

Berechnung von h_einsp (Einspeisezone) und h_klemm (Klemmenzone) analog zur Kabeleinführungszone. Erweiterung für Standschrankvarianten.

  • h_einsp-Berechnung
  • h_klemm-Berechnung
  • Standschrank (h_zug_ke ≈ 35 mm)
In Vorbereitung
03
Geplant

TE-Berechnung & Reihenkapazität

Berechnung der Teilungseinheiten pro Reihe und Gesamtkapazität der Montageplatte auf Basis von Schrankgeometrie und Kanalbreiten.

  • TE pro Reihe: (MP-Breite − b_kanal_v) / 18
  • HE Reihen Variante 1 & 2
  • Datenbankabfrage via sql.js

Softwarearchitektur

Clientseitige Anwendung ohne Server – kompatibel mit GitHub Pages.

Datenbankstrategie

Excel (.xlsx) als Pflegewerkzeug · SQLite (.db) als Produktivdatenbank · Konvertierung via Python · Zugriff im Browser über sql.js (WebAssembly, kein Server).

python convert.py # xlsx → SQLite

Datenbankschema (Auswahl)

Tabelle schraenke_wand mit geometrischen Feldern, Preisfeldern und DL-Feldern. Jeder Datensatz hat Aktiv-Flag und Preisdatum für revisionssichere Pflege.

id aussen_breite_mm h_ke_mm preis_gesamt_schrank_eur aktiv preis_datum
